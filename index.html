🤖 Como usar pelo Claude RECOMENDADO

  • 1. Acesse claude.ai e faça login.
  • 2. Inicie uma conversa nova e anexe o arquivo do Guia ETP (GUIA_ETP_IA_A5_v2.pdf) clicando no ícone de clipe.
  • 3. Digite: "Abra isso como artefato e use como gerador" — o Claude abrirá este gerador diretamente na conversa.
  • 4. Preencha os Dados Gerais (Passo 1) e, em cada bloco (Passo 2), use o botão 🤖 Gerar com IA — o texto é redigido diretamente aqui, sem precisar copiar e colar em outro lugar.
✅ Vantagens do Claude:
• Gera o texto de cada bloco direto nesta página, sem precisar copiar/colar
• Botão ⚡ Gerar Todos com IA (Passo 3) redige todos os 13 elementos de uma vez
• Exporta o ETP completo em arquivo .docx no padrão do ETP Digital, pronto para revisar e assinar

🌐 Como usar no navegador (ChatGPT, Gemini…) SEM IA INTEGRADA

  • 1. Preencha os Dados Gerais (Passo 1) e os campos de cada bloco (Passo 2) normalmente.
  • 2. Clique em ⚙ Gerar Prompt em cada bloco e depois em 📋 Copiar Prompt.
  • 3. Ou use o ⚡ Gerar Prompt Único (Passo 3) para compilar o ETP inteiro em um só prompt.
  • 4. Cole em uma IA externa (ChatGPT, Gemini, Copilot…), copie a resposta e cole no campo verde do bloco — ela será distribuída automaticamente entre os elementos.
⚠️ Atenção ao colar no assistente:
• Evite colar o Prompt Único de uma vez em assistentes mais simples — prefira enviar no máximo 2 blocos por vez (Passo 2) para reduzir erros e alucinações da IA.
• Aguarde a resposta e revise antes de enviar o próximo grupo.

III·V

    
Requisitos da Contratação + Levantamento de Mercado
Elementos III e V — complementares, elaborados juntos

    NÃO OBRIGATÓRIO
▼

  

    
📌 Requisitos da Contratação + Levantamento de MercadoSão complementares e devem ser coerentes: o levantamento mostra o que o mercado oferece; os requisitos definem o filtro mínimo. Requisitos fechados demais podem configurar direcionamento indevido.

Elementos IV e VI — a quantidade fundamenta o valor

    OBRIGATÓRIO

📌 Estimativa das Quantidades + Estimativa do ValorO elemento IV indica quanto contratar (com memória de cálculo); o VI aponta quanto deve custar (com fonte e data da pesquisa). Devem ser coerentes entre si.
